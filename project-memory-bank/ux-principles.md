# UX Principles

## Style & Aesthetics
- **Theme**: Slate/Dark-mode by default. Focus on premium glassmorphism elements, clean layout structures, and high contrast accents (violet/indigo text and buttons).
- **Typography**: Outfit for headings, Inter for body copy.
- **Calmness**: Avoid clutter. Dashboards should be clean, content-focused, and display clear, actionable options.

## Interactivity
- **Micro-animations**: Subtle transitions for button states, card hover, list sorting, and dashboard navigation.
- **Feedback**: Immediate validation states on inputs. Clear, non-intrusive toast notifications for success/error alerts.
- **Predictability**: Elements behave consistently. A button that executes an agent action has clear loading states so the user is never left guessing.
